## MACD の位置づけ（必須）
- MACD は **モメンタムとトレンド変化の検出補助**
- 主軸トレンドは SMA
- 過熱判定は RSI
- MACD は「変化の兆し」を担う

## 必須記載内容
- Layer1:
  - MACD ライン
  - シグナルライン
  - ヒストグラム
- Layer2:
  - 乖離量（正規化）
  - 0 ラインとの位置関係
  - ヒストグラム増減速度
- Layer3:
  - 「勢い増加」「減速」
  - 「トレンド加速の兆候」
  - 「反転初動の可能性」

## 他インジケーターとの関係
- SMA と同方向 → 信頼度上昇
- RSI と逆行 → エントリー見送り理由に使う
- MACD 単独での売買判断は禁止
