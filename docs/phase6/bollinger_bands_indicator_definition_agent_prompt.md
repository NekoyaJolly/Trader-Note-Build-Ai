## BB の位置づけ（必須）
- BB は **ボラティリティ・価格位置系インジケーター**
- トレンド判断の主軸にはしない
- 「拡張 / 収縮」「バンド内外の位置」を扱う

## 必ず含める観点
- Layer1: 上限・下限・ミドルバンド、σ、期間
- Layer2:
  - バンド幅（正規化必須）
  - 価格のバンド内位置（%B）
  - バンド拡張 / 収縮状態
- Layer3:
  - 「拡散局面」「収束局面」
  - 「バンドウォーク」「バンドタッチ」
- SMA / RSI との関係
  - SMA が上向き + 上バンド張り付き = トレンド継続示唆
  - RSI だけで逆張りしないための補助指標

## 禁止事項
- BB 単体で売買判断を完結させない
- ±2σ を絶対的な売買ラインとして断定しない
